
== Agora App Builder ==
Instructions to run your project:
1. Open a terminal inside this folder
2. Run - npm install && cd advancedapiexamples && npm install && cd .. (You need NodeJS v16.x installed on your system)
3. To checkout customization examples go to advancedapiexamples/customization/api 
4. To see preview of the customization examples go to advancedapiexamples/customization/index.tsx
5. Uncomment example that you want to check and comment out rest of lines
6. Run - npm start 
7. Select build through the CLI menu
8. Build for any supported platform
